# Sharmarke Abdi Abdullahi - Professional Portfolio

## Tech Stack
- Node.js + Express.js
- EJS templating
- PostgreSQL
- Bootstrap 5
- JavaScript (vanilla)

## Setup
1. Create PostgreSQL database
2. Copy `.env.example` to `.env` and fill credentials
3. Run `npm install`
4. Run `npm start`
5. Visit `http://localhost:3000`

## Admin Access
- `/admin/login`
- Use credentials from `.env`
